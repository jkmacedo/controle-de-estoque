# Objetvo

Esse programa tem por objetivo ajudar colaboradores da área de almarifado no gerenciamento dos materiais;

---

# Sistema de Controle de Estoque

Sistema desenvolvido em **Python** para gerenciamento de materias, focado em lógica backend e persistência de dados com **SQL**.

---

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.14
- **Banco de Dados:** SQL (Pasta `/data`), (Pasta `/dao`)
- **Arquitetura:** Separação de responsabilidades.
- **Interface gráfica:** CustomTkintir
